Please use download.mozilla.org to retrieve the latest Firefox ESR. 

For example, if you wish to download the US English installer for Windows (32bit)

   https://download.mozilla.org/?product=firefox-esr-latest&os=win64&lang=en-US

This could be pasted into the location bar of a browser, or used with curl or wget,  e.g. 
wget -O FirefoxSetup.exe "https://download.mozilla.org/?product=firefox-esr-latest&os=win64&lang=en-US"

For other operating systems replace 'os=win' with:
   Windows 64bit              os=win64
   OS X                       os=osx
   Linux x86_64               os=linux64
   Linux i686                 os=linux
